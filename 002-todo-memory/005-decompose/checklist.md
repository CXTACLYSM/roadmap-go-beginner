## Самопроверка через рисунок

На листе бумаги нарисуй три блока друг под другом:

```
┌─────────────────┐
│      main       │  ← REPL, парсинг
└────────┬────────┘
         │ вызывает
┌────────▼────────┐
│  handle-функции │  ← обработка команд, вывод
└────────┬────────┘
         │ вызывает
┌────────▼────────┐
│    TaskList     │  ← бизнес-логика, состояние
└─────────────────┘
```

Сможешь ли ты для **каждого** из своих файлов/функций сказать, к какому блоку он относится? Если да — ты понял, что такое **слои**. Это понимание — самое ценное в этом модуле.
