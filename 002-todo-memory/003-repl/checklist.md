# Модуль 3. REPL — цикл команд — чек-лист готовности

## Поведенческие маркеры

- [ ] Программа запускается и выводит приветствие
- [ ] Перед каждым вводом выводится приглашение `> `
- [ ] Команда `add <текст>` добавляет задачу с уникальным ID
- [ ] ID начинаются с 1 и инкрементируются (1, 2, 3, ...)
- [ ] Команда `list` выводит все задачи через `String()`
- [ ] Команда `list` для пустого списка выводит «Список пуст»
- [ ] Команда `exit` корректно завершает программу
- [ ] Неизвестная команда выводит сообщение, программа продолжает работать
- [ ] Пустой ввод (просто Enter) — игнорируется, программа продолжает работать
- [ ] Используется `bufio.Scanner`, **не** `fmt.Scan`
- [ ] Используется `strings.SplitN(..., 2)`, не `Split`
- [ ] Используется `strings.TrimSpace` для очистки ввода
- [ ] Слайс `tasks` и счётчик `nextID` объявлены **до** цикла
- [ ] Код закоммичен
