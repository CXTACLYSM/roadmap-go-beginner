# Модуль 4. Изменение и удаление — чек-лист готовности

## Поведенческие маркеры

- [ ] Реализована функция `findIndex(tasks []Task, id int) int`
- [ ] `findIndex` возвращает `-1`, если задача не найдена
- [ ] Команда `done <ID>` отмечает задачу как выполненную
- [ ] Команда `done` без аргумента → подсказка по использованию
- [ ] Команда `done` с буквами вместо ID → понятная ошибка
- [ ] Команда `done` с несуществующим ID → «Задача не найдена»
- [ ] Команда `remove <ID>` удаляет задачу
- [ ] После `remove` в слайсе становится на один элемент меньше (`len(tasks)` уменьшается)
- [ ] Команда `remove` обрабатывает все те же ошибки, что и `done`
- [ ] ID удалённых задач **не переиспользуются** при следующем `add`
- [ ] Все команды из модуля 3 продолжают работать
- [ ] Код закоммичен
