## Подготовка к модулю 2

В модуле 2 ты сделаешь первые **GET-эндпоинты**, которые возвращают JSON: список задач и одну задачу по ID. Тебе понадобится:

- **Path patterns** Go 1.22+ для парсинга `/tasks/{id}` без внешних библиотек
- **JSON-сериализация ответа** — снова `encoding/json`, на этот раз результат уходит клиенту, а не в файл
- **Заголовок `Content-Type: application/json`** — чтобы клиент знал, что в ответе именно JSON

К этому моменту у тебя должен быть установлен **Go 1.22 или новее**. Проверь:

```bash
go version
```

Если версия меньше — обнови. Без этого path patterns не сработают.
