## Подготовка к модулю 4

В модуле 4 ты переписываешь **весь** `TaskList` на SQL. Все методы (`Add`, `GetByID`, `All`, `Update`, `Remove`) будут использовать `database/sql` для работы с PostgreSQL вместо in-memory слайса.

Главные новые вещи:

- **`db.Query`** — запрос, возвращающий **много** строк (для `All`)
- **`*sql.Rows`** и итерация через `rows.Next()` + `rows.Scan()`
- **`db.Exec`** — запросы без результата (для `Update`, `Remove`)
- **Параметры запроса** через `$1`, `$2` — защита от SQL injection
- **`RETURNING`** для получения присвоенного `id` после INSERT
- **`sql.ErrNoRows`** — особая ошибка, означающая «строка не найдена»

К концу модуля 4 у тебя будет полная замена in-memory `TaskList`, готовая к интеграции с HTTP-сервером (модуль 6).
